# YouTube Homepage Clone

This project is a front-end replica of the YouTube homepage built using HTML and CSS as part of the SuperSimpleDev HTML & CSS course.

## Features

- Responsive YouTube-style homepage
- Navigation bar
- Sidebar
- Video cards
- Search bar
- Channel profile images
- Hover effects and tooltips
- CSS Grid and Flexbox layout
- Clean and organized code

## Technologies Used

- HTML5
- CSS3

## Learning Outcomes

- Semantic HTML
- CSS Grid
- Flexbox
- Positioning
- Responsive Web Design
- Hover Effects
- CSS Transitions
